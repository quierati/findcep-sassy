O que FindCEP oferece ao seu negócio?

A API FindCEP é a mais ágil e completa solução
de localizacação de codigo postal e endereços no BRASIL.

Base integrada com os Correios

FindCEP oferece busca e tratamento de CEPs (Código Enderecamento Postal) e Endereços para o seu negócio online.

Classificação de Localidades

Só o FindCEP inclui a estrutura do cep contendo localidades não codificadas (CEP Unico) e localidades codificadas como caixa postal comunitaria.

Base de CEP atualizada

Nossa base de CEPs é atualizada mensalmente e integrada automaticamente com os correios.

Códigos dos Municípios - IBGE

Nossa base é sincronizada com a última versão da base disponibilizada pelo IBGE , Incluindo corretamente os códigos dos distritos e sub-distritos.

Taxa de Atualização

Em media são disponibilizados pelos correios atualizacoes com cerca de 10 mil endereços/cep por mês.

Arquitetura

O FindCEP fica hospedado na Cloud AWS, utilizando as melhores praticas do mercado. O produto foi desenvolvido para performar durante as Black-friday's
